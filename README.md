# K3s Fleet GitOps Setup

This repository contains the GitOps configuration for your k3s cluster managed by Fleet.

## Repository Structure

```
k3s-server/
├── fleet.yaml              # Fleet configuration for this repository
├── clusters/               # Cluster-specific configurations
│   └── local/             # Local k3s cluster
└── apps/                  # Application deployments
    └── example/           # Example application
```
